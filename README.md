# Spacecraft-Camera

Features:  
 - Directional movement  
 - Strafing  
 - Rotation (pitch, yaw, roll)  
 - Zoom (Fov)

Controls:
 - Camera forward direction: Mouse
 - Forward: W
 - Backward: S
 - Strafing Left: A
 - Strafing Right: D
 - Rolling Right: E
 - Rolling Left: Q
 - Strafing UP: Z
 - Strafing Down: Y
 - Zoom in: NUMPAD+
 - Zoom out: NUMPAD-
